In this tutorial we are going to get basic understanding about Spring Boot application. We will have a hands on experience creating CRUD operation. CRUD stands for Create Read Update Delete, witch are the most common HTTP methods: GET, POST, PUT, and DELET respectively. 

For storing data we will use H2 database, a lightweight Java database. wich can be embedded in Java application. H2 can be run as in memory or persist on disk.
